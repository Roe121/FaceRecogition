EasyFaceRec

This is a simple face recognition application that allows real-time face detection and recognition from images, created with the help of the face_recognition library.

Requirements:
- Python 3.11
- OpenCV (cv2)
- dlib (for Python 3.11)
- face_recognition
- Pillow

Installation:
1. Python 3.11 from the official website
2. pip install opencv-python
3. py -m install dlib-19.24.1-cp311-cp311-win_amd64.whl (dlib file can be found in \FaceRecogition\Dlib)
4. pip install Pillow

Starting the application:
1. Run the main.py script:

Usage:
- Add your personal known faces to the "images" folder.
- Click on the "Realtime Detection" button to start real-time face detection.
- Click on the "Image Detection" button to select an image and detect faces in it.

Note: 
-Make sure to put your images inside the "images" folder.
-Test images provided in : \FaceRecogition\test images
